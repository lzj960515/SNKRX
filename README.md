# SNKRX中文版

[SNKRX](https://store.steampowered.com/app/915310/SNKRX/) 是一款街机射击类Roguelite游戏，你可以控制一条由自动攻击附近敌人的英雄组成的蛇。
组合不同的英雄以解锁职业加成并创建独特的构建，指引你无敌的队伍冲破无尽的敌人浪潮。

### 运行方法

下载此仓库，进入目录后运行 `engine/love/love.exe --console .` 即可。无需启动Steam。
